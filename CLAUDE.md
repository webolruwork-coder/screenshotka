# CLAUDE.md — Скриншотилка

Минималистичное приложение для снимков экрана для macOS (Swift, меню-бар, своё выделение области, плавающее превью, редактор аннотаций). Подробности — в `README.md`.

## Правила проекта

### Интерфейс
- Делаем **максимально дружелюбный** интерфейс для пользователя.
- Интерфейс должен быть **максимально нативным к macOS Mojave**: используем системный UI/UX-kit и системные компоненты и решения, а не самописные аналоги, где это возможно.
- Используем **best practices, паттерны, компоненты и навигацию** из [Apple Human Interface Guidelines](https://developer.apple.com/design/human-interface-guidelines/).

### Подход к работе над задачей
Перед тем как выполнять задачу:
1. **Проверь, насколько ты её понял** — переформулируй суть и убедись, что понимание совпадает с ТЗ.
2. **Подумай, чего не хватает** и что не учтено в задаче или ТЗ.
3. **Продумай лучшее решение** — так, чтобы пользователю было удобно.
