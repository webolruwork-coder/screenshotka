# Скриншотилка

Минималистичное приложение для снимков экрана в духе CleanShot X — живёт в меню-баре, со своим выделением области, плавающей превью-плашкой и встроенным редактором аннотаций.

## Возможности

- **Снимок области** `⌃⇧4` — затемнённый оверлей, перетаскивание рамки, живой размер в пикселях, ручки для точной подгонки, стрелки клавиатуры (с Shift — шаг 10 px), `⏎` — снять, `Esc` — отмена.
- **Снимок окна** `⌃⇧5` — подсветка окна под курсором, клик — снимок.
- **Весь экран** `⌃⇧3` — экран под курсором.
- **Плавающая превьюшка** в углу после снимка: навёл — появляются действия (аннотации / копировать / сохранить / закрыть), можно перетащить файл прямо в другое приложение, авто-скрытие через 6 c.
- **Редактор аннотаций**: стрелка, линия, прямоугольник, овал, карандаш, маркер, размытие (пикселизация), текст; палитра, толщина, отмена/повтор, копирование и сохранение.
- **Меню-бар**: папка снимков, переключатели (буфер обмена, превью, автосохранение, звук), выбор папки сохранения.

Снимки по умолчанию копируются в буфер и сохраняются в `~/Pictures/Screenshots`.

## Сборка

Нужен Xcode (SDK macOS 14+). Если ругается на лицензию — один раз `sudo xcodebuild -license accept`.

```bash
./setup-signing.sh   # один раз: стабильная подпись, чтобы права не слетали при пересборке
./build.sh
open "build/Скриншотилка.app"
```

## Первый запуск и разрешение

При первом снимке macOS попросит разрешение **Запись экрана**
(Системные настройки → Конфиденциальность и безопасность → Запись экрана).
Включите «Скриншотилка» и сделайте снимок снова.

Благодаря `setup-signing.sh` подпись стабильна между сборками, поэтому
разрешение выдаётся **один раз** и не сбрасывается при пересборках.
Без неё `build.sh` подписывает ad-hoc и сбрасывает право каждую сборку
(тогда доступ придётся выдавать заново после каждой сборки).

## Архитектура

| Файл | Назначение |
|------|-----------|
| `main.swift` / `AppDelegate.swift` | точка входа, меню-бар, горячие клавиши, оркестрация |
| `HotkeyManager.swift` | глобальные хоткеи (Carbon) |
| `ScreenCapturer.swift` | захват через ScreenCaptureKit (`SCScreenshotManager`) + список окон |
| `SelectionOverlay.swift` | оверлей выделения области/окна |
| `PreviewPanel.swift` | плавающая превью-плашка |
| `EditorView.swift` / `EditorWindowController.swift` / `Annotation.swift` | редактор |
| `ImageStore.swift` | сохранение, копирование, экспорт |
| `Settings.swift` / `Theme.swift` | настройки и оформление |

> Захват — на ScreenCaptureKit; собственное приложение исключается из кадра
> (`excludingApplications`), поэтому оверлей и плашки не попадают в снимок.
> Сборка идёт напрямую через `swiftc` (бандл `.app` собирается вручную в `build.sh`).
> Экспорт редактора рендерится через flipped `NSView` + `cacheDisplay` в 1:1 — без
> масштабирования Retina и без зеркалирования.
